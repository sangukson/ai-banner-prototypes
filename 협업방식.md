# AI 배너 프로토타입 협업 방식

## 프로세스

1. **탐색** — 디자이너 + Claude로 여러 시안 빠르게 제작
2. **결정** — 내부 논의 후 최종안 확정
3. **전달** — Linear 티켓에 최종 프로토타입 링크만 첨부
4. **개발** — 개발자(+ Claude Code)가 링크 열어 구조·플로우 파악 후 구현

> 프로토타입은 **플로우·구조 참고용**. 색상·간격·폰트는 핸드오프(Figma) 기준.
